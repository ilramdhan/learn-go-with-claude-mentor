# 📁 Assets — Fase 5: gRPC & Protobuf

Folder ini menyimpan **aset visual** yang mendukung materi Fase 5.

## 📋 Jenis Aset yang Tersimpan di Sini

Aset dapat berupa:
- 🖼️ **Diagram arsitektur** (PNG/SVG)
- 📊 **Infografis** konsep-konsep penting
- 📸 **Screenshot** tools, UI, atau output terminal
- 🗺️ **Mindmap** dan ringkasan visual

## 🎨 Daftar Aset yang Direkomendasikan

Untuk melengkapi materi Fase 5, disarankan menambahkan:

> diagram gRPC lifecycle, HTTP/2 vs HTTP/1.1 comparison, proto schema diagram

## 📥 Cara Berkontribusi Aset

1. Buat diagram menggunakan tools seperti:
   - [draw.io](https://draw.io) — gratis, export ke PNG/SVG
   - [Excalidraw](https://excalidraw.com) — sketsa-style, bagus untuk diagram informal
   - [Mermaid.js](https://mermaid.live) — diagram dari kode Markdown
   - [PlantUML](https://plantuml.com) — diagram dari teks

2. Simpan dengan naming convention: `fase05-[nama-konsep].[png|svg|jpg]`
   - Contoh: `fase05-architecture-diagram.png`
   - Contoh: `fase05-sequence-flow.svg`

3. Buat PR dengan aset baru dan update README ini

## 💡 Tips Membuat Diagram yang Baik

- Gunakan warna yang konsisten (biru untuk service, hijau untuk DB, oranye untuk queue)
- Tambahkan judul dan legenda yang jelas
- Export resolusi tinggi (minimal 1200px wide untuk PNG)
- SVG lebih baik dari PNG karena scalable
