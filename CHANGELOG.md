## [1.0.0] — 2025-03-16

### Added — Initial Release 🎉

#### Materi Lengkap (Fase 1–6)
- **Fase 1: Go Fundamentals** — 13 modul (setup, variables, types, control flow, functions, slices, maps, structs, pointers, errors, packages, defer/panic/recover)
- **Fase 2: Go Intermediate** — 13 modul (interfaces, type assertions, goroutines, channels, select, sync, context, generics, testing, file I/O, JSON, time, idioms)
- **Fase 3: Gin Framework & REST API** — 13 modul (routing, request handling, middleware, GORM, repository pattern, JWT, error handling, validation, file upload, Viper, Swagger, testing)
- **Fase 4: Clean Architecture** — 12 modul (layers, domain entity, value object, repository interface, use case, delivery layer, dependency injection, unit testing, graceful shutdown, health check)
- **Fase 5: gRPC + Protocol Buffers** — 13 modul (protobuf syntax, toolchain, unary RPC, server/client/bidi streaming, error handling, interceptors, auth, reflection, clean arch integration, gRPC gateway)
- **Fase 6: Domain-Driven Design** — 12 modul (DDD philosophy, ubiquitous language, entity, value object, aggregate, domain events, domain services, application services, bounded context, CQRS, event sourcing intro)
